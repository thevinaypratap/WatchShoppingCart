## Run Locally

Clone the project

```bash
  git clone https://github.com/Mukeshkumar697
```

Go to the project directory

```bash
  cd React_E-Commerce (E- Fertilizer)
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
